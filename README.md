## 🗂️ 프로젝트 목록

### 냉장고 시뮬레이터 [→ 자세히 보기](refrigerator-simulator\README.md)
원래는 실제 하드웨어(냉장고 + 센서 + 쿨러)로 전부 구현하고 싶었지만, 시간·비용·공간의 제약이 있어서. 그래서 파이썬 시뮬레이터를 먼저 만들어 시스템의 전체 구조와 동작 원리를 이해하기 위해서 만듦

## 🛠️ 기술 스택

### 하드웨어
- Arduino (Uno, Nano, ESP32 등)
- 각종 센서 (온도, 습도, 거리, 등)
- 액추에이터 (모터, LED, 부저 등)

### 소프트웨어
- C/C++ (Arduino)
- Python (시뮬레이션, 데이터 분석)
- MQTT, HTTP (IoT 통신)
