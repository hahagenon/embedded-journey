# 🧊 냉장고 시뮬레이터


**왜 만들었나?**  
하드웨어를 구하기 어려워서 파이썬으로 가상 시뮬레이터를 먼저 구현을 통해 원리 이해

**시스템 구성**

**[현재 구현]**
- **Python 시뮬레이터**: 
  - 가상 냉장고 물리 엔진 (온도 상승/하강 시뮬레이션)
  - 가상 온도 센서 (센서 고장 시뮬레이션 포함)
  - 가상 액츄에이터 (압축기, 댐퍼)
  - 실시간 GUI (온도 모니터링, 그래프)
  - 제어 로직 (Hysteresis)

- **Arduino (하드웨어)**: 
  - 가상 쿨러 제어 (on / off)

**통신 구조**
```
[Python 시뮬레이터]  ←─ 시리얼(UART) ─→  [Arduino]
  ├─ 온도 데이터 전송 →                      ├─ 온도 수신
  ├─ 제어 명령 수신 ←                        └─ 쿨러 제어 [on / off]
  └─ GUI 시각화
```

**프로젝트 영감**  
ROS2 학습 중 QoS의 Deadline, Lifespan, Liveliness 개념("신호가 끊기거나 살아있음을 감지")이 흥미로웠는데, ROS2는 너무 무거워서 시리얼 통신으로 더 가볍게 비슷한 실시간 제어 시스템을 구현해보고 싶었음.

**학습 목표**
1. 시뮬레이터로 제어 로직 완성
2. 시리얼 통신 프로토콜 설계
3. 실제 하드웨어 연결 (Arduino)
4. 센서 고장 대응 로직 검증

**개발 환경**  
Ubuntu 노트북 + Python + (향후) Arduino + 시리얼 통신

---

냉장고의 온도 제어 시스템을 시뮬레이션하는 프로젝트입니다.  
코드를 단계적 구현한것을 차례대로 올립니다.
이는 제가 GUI와 시뮬레이터를 작성시 코드 이해와 과정을 기록하기 위함입니다.
또한 실제 냉장고가 어떻게 온도를 유지하는지 이해하고, 제어 로직을 직접 구현해봅니다.

### 🧊 냉장고 시뮬레이터
냉장고의 온도 제어 시스템을 시뮬레이션하는 프로젝트

**주요 기능:**
- 실시간 온도 모니터링 (냉장실/냉동실)
- 압축기 및 댐퍼 자동 제어 (Hysteresis 로직)
- 실시간 그래프 시각화
- 센서 고장 시뮬레이션 및 긴급 정지
- 이벤트 로그 시스템

**기술 스택:**
- Python (tkinter, matplotlib)
- 물리 시뮬레이션
- 멀티스레딩

**학습 목표:**
- 센서-액추에이터 제어 로직 이해
- 실시간 시스템 시뮬레이션
- GUI 기반 모니터링 시스템 구현
