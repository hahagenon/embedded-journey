## ✨ 주요 기능

### 1. 실시간 온도 모니터링
- 냉장실과 냉동실의 온도를 실시간으로 표시
- 목표 온도와 현재 온도를 함께 확인
- 센서 상태 모니터링

### 2. 자동 제어 시스템
- **압축기 제어**: 냉동실 온도에 따라 자동 ON/OFF
- **댐퍼 제어**: 냉장실 온도 조절
- **Hysteresis 로직**: 떨림 현상 방지 (±2°C 범위)

### 3. 실시간 그래프
- 온도 변화 추이를 시각적으로 확인
- 목표 온도 기준선 표시
- 최대 100개 데이터포인트 유지

### 4. 장애 시뮬레이션
- 냉장실 센서 고장
- 냉동실 센서 고장
- 제어기(Arduino) 통신 두절
- 자동 복구 (5초 후)
- 긴급 정지 기능

### 5. 로그 시스템
- 모든 이벤트 기록 (타임스탬프 포함)
- 압축기/댐퍼 상태 변화 로그
- 장애 발생 및 복구 로그
